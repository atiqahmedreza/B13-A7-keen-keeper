# KeenKeeper

KeenKeeper is a modern friendship relationship tracker built with Next.js. It helps you stay connected with the people who matter most by showing which friends need attention, logging your interactions, and visualizing your relationship history in a clean and friendly interface.

## Project Overview

In today’s busy life, it is easy to lose touch with important people. KeenKeeper makes relationship care simple by giving you a dashboard to:

- track your friends and their current status
- log calls, texts, video chats, and meetups
- monitor who is overdue for a check-in
- review your interaction timeline and statistics

## Features

- Responsive friend dashboard with status cards and tags
- Friend detail pages with contact goals and next due dates
- Quick check-in actions for instant interaction logging
- Timeline view for recent conversations and activities
- Stats page with visual charts for interaction insights
- Smooth UI experience with loading states and toast notifications

## Technologies Used

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Recharts
- Zustand
- Sonner
- lucide-react

## Getting Started

```bash
git clone https://github.com/atiqahmedreza/B13-A7-keen-keeper.git
cd B13-A7-keen-keeper
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

## Build for Production

```bash
npm run build
npm run start
```

## Deployment

For Netlify, use:
- Build command: `npm run build`
